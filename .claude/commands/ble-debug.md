# BLE デバッグ解析

以下の手順でBLE接続問題を解析してください。

1. `lib/data/datasources/ble_datasource.dart` の最新実装を確認
2. `flutter logs` の直近エラーを取得
3. 以下の観点で問題を分類:
   - アプリ側の問題（状態管理・タイムアウト設定）
   - 通信の問題（RSSI・パケットロス）
   - デバイス側の問題（韓国のThe Clockのファームウェア）
4. 原因と修正コードを提示
5. リトライ・リカバリ設計の改善案があれば合わせて提示
